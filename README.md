# Galaxy Store

How to Run

1. Clone the repository.
2. Use node version `v18`.
3. Install the dependencies `npm install`.
4. Build the aplication `npm run build`.
5. Run `npm run start`.

## Technologies Used

- Angular
- Ngrx
- Bootstrap
